# GetPopularMusic-Swift
https://rss.itunes.apple.com/ja-jp のRSSから最新の人曲を取得するSwiftプログラム

## Get Started
1. `pod install`

## Architecture
- MVVM

## Installing
- `RxSwift`
- `Alamofire`
- `SwiftyJSON`
